> **AI-powered professional networking platform** with glassmorphism design, real-time messaging, and intelligent job matching.

##  Features

 **Professional Networking** •  **AI Job Matching** •  **Real-time Messaging** •  **Analytics Dashboard** •  **Company Management** •  **PWA Support**

##  Tech Stack

**Frontend:** React 18 + TypeScript + Vite + Tailwind CSS + Framer Motion + Zustand  
**Backend:** Django 4.2 + DRF + JWT + SQLite + OpenAI + ML/AI Services  
**Deployment:** Vercel + Render (Free Tier) • **Cost:** $0/month

##  Quick Start

```bash
# Clone & Setup
git clone https://github.com/reuben-idan/CareerOpen-App.git
cd CareerOpen-App

# Frontend
cd frontend && npm install && npm run dev

# Backend (new terminal)
cd backend && pip install -r requirements.txt && python manage.py migrate && python manage.py runserver
```

**Prerequisites:** Node.js 18+ • Python 3.9+
